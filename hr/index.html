3. 신규 후보자 페이지 추가 방법

  1. hr/interviewees/template.html 파일을 복사합니다.
  2. 파일명을 후보자 영문명 또는 한글명 영문표기로 저장합니다. 예: kimyounghee.html
  3. 템플릿 안의 후보자명, 포지션, 평가 점수, 의사결정 내용을 수정합니다.
  4. hr/index.html의 인터뷰이 평가자료 섹션에 후보자 카드를 추가합니다.
  5. GitHub에 업로드 후 Pages URL에서 정상 노출되는지 확인합니다.
주의: GitHub Pages의 정적 비밀번호 게이트는 강력한 보안이 아닙니다. 민감 자료는 private repository, 별도 인증 서버, Notion 권한관리, Google Drive 권한공유 등을 사용하세요.
